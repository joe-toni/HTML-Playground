A <figcaption> is a tag usually used within a figure parent along with an <img&rt tag to indicate that the text contained is a caption for the preceding image. It has no special renduring but can be styled by name in css and is expected to either be the first or last child within the parent <figure&rt tag.

The <div&rt tag is a generic container meant to be used in a multitude of ways depending on the aim of the author. It has no special renduring unless specified through styling and can be the parent or child of most any other HTML element but is expected to be used only when no other semantic element is appropiate.

A <blockquote> is used to indicate that the text contained within is an extended quote. It is typically used along with a <cite&rt and <p&rt with the cite tag providing a source for the quote and a the paragraph tag actually containing the disired text. When rendured basically the blockquote tags will add indentation to the paragraph in order to indicate a quotation but this can be adjusted through styling the margin.
